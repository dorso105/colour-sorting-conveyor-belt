# colour-sorting-conveyor-belt
this is a conveyor belt that uses colour items pending for where it will go.
I made this so i can sort out my filament poop in colour so i can grind it down to melt it using custom moulds.




If the code does not work please try using the following.


common error readings may be as following.

ModuleNotFoundError: adafruit_tcs34725
→ install with: pip3 install adafruit-circuitpython-tcs34725

I2C not working → enable it: sudo raspi-config → Interface Options → I2C → Enable



The 3d models are to a 0.001 (±0.001) 
please scale them or change tolorance accordingly depending on 3dprinter.
